# Public Assets yang Diperlukan

File-file berikut perlu ditambahkan ke folder `public/`:

1. **favicon.ico** - Icon untuk browser tab
2. **pwa-192x192.png** - Icon PWA ukuran 192x192 pixels
3. **pwa-512x512.png** - Icon PWA ukuran 512x512 pixels
4. **apple-touch-icon.png** - Icon untuk iOS (opsional)
5. **mask-icon.svg** - Mask icon untuk Safari (opsional)

Anda dapat menggunakan tool online seperti [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator) atau membuat icon secara manual menggunakan desain grafis.

Untuk sementara, aplikasi akan tetap berfungsi tanpa icon, namun fitur PWA install mungkin tidak optimal tanpa icon yang sesuai.
